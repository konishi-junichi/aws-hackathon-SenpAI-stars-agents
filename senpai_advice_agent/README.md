# SenpAI Advice Agent

新入社員のメンタルサポートとキャリア・自己成長支援を専門とするAIエージェントです。

## 🎯 主な機能

### メンタルサポート
1. **傾聴と共感**: 悩みや不安を親身に聞き、共感的に対応
2. **励ましと勇気づけ**: 前向きな気持ちになれるような励ましを提供
3. **心理的安全性**: 安心して相談できる環境を提供
4. **ストレス管理**: 仕事のストレスや人間関係の悩みへのアドバイス
5. **ワークライフバランス**: 健康的な働き方の提案

### キャリア・自己成長支援
1. **キャリア設計**: 将来のキャリアパスの相談と提案
2. **スキル開発**: 必要なスキルの特定と習得方法の提案
3. **自己啓発**: 成長のための学習計画や目標設定のサポート
4. **コンサルティングスキル**: 論理的思考、問題解決、コミュニケーション能力の向上
5. **目標達成**: SMART目標の設定と達成のためのアクションプラン

## ⚙️ 技術仕様
- **モデル**: Amazon Nova Micro (`us.amazon.nova-micro-v1:0`)
- **フレームワーク**: LangGraph + Amazon Bedrock
- **ランタイム**: Amazon Bedrock AgentCore Runtime
- **温度設定**: 0.7（共感的で温かい対応のため）

## 📝 使用例

### 入力例
```
新入社員として入社したばかりですが、周りについていけるか不安です。
どうすれば自信を持って仕事に取り組めるでしょうか？
```

### 出力例
```
## 新入社員の不安、とてもよく分かります！ 😊

### まず知っておいてほしいこと 💡
新入社員が「ついていけるか不安」と感じるのは、**とても自然で健全な反応**です。
これは成長意欲の表れでもあります。

### 自信を育てる3つのステップ 🌱

#### 1. 小さな成功を積み重ねる
- 毎日1つでも「できた！」を見つける
- 分からないことを質問できた → 成功
- 新しいことを覚えた → 成功

#### 2. 周りとのコミュニケーションを大切に
- 先輩や同僚に積極的に質問する
- 「教えていただき、ありがとうございます」の気持ちを伝える

#### 3. 成長の記録をつける
- 週末に「今週学んだこと」を振り返る
- 1ヶ月前の自分と比較してみる

### 今日からできること ✨
まずは「質問リスト」を作ってみませんか？
分からないことを整理するだけで、不安が具体的な行動に変わります。

あなたの成長を応援しています！ 🌟
```

## 🎯 目的

新入社員が安心して成長できる環境を提供し、メンタル面とキャリア面の両方から包括的にサポートします。
